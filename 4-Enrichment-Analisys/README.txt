================================
Enrichment-pathway-targets-all.R
================================

Performs the miRNA enrichment analysis in pathways considering
all miRNA targets.

Output:

A CSV file containing the scored list of miRNA
A plot with the top 30 with p<0.01 scored miRNA


=============================
Enrichment-pathway-targets-mirtarbase.R
=============================

Performs the miRNA enrichment analysis in pathways considering
miRNA targets from miRTarBase.

Output:

A CSV file containing the scored list of miRNA
A plot with the top 30 with p<0.01 scored miRNA
