mirna_total_targets.csv
==================================

For each miRNA reports its total target count (fist column considers targets from all DBs second column
only targets from targetmine).
The last row contains the total count.

TargetHUB dump considering targets in all databases. One file for each mirna.

==================================

==================================

==================================
targets-mirtarbase-targeted-genes.json

List of genes that appear at least once as targets in targets-mirtarbase
